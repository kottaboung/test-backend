



## Description
testing backend for test frontend next

## Project setup

```bash
#install package
$ npm install

#generate component with resource
$ nest g r <component-name>
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
